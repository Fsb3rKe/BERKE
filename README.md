# berke
a music player.

## Contribute
Clone the project.
```sh
git clone https://github.com/fsb3rke/berke
```
Move to `berke` folder.
```sh
cd ./berke
```
Create build folder with cmake
```sh
cmake -DCMAKE_BUILD_TYPE=Release -S . -B build -G"Unix Makefiles"
```
Build
```sh
make
```
